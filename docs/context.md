This is a production-level e-commerce platform for a local chakki-based healthy food business.

Tech Stack:
- Backend: Node.js, Express, MongoDB
- Auth: JWT + OTP
- Payments: Razorpay + COD
- Invoices: GST-compliant PDF
- Frontend: Next.js
- Admin Panel: Separate UI

Architecture:
- Modular monolith, scalable to microservices
- Admin controls pricing and products
- Future mobile app support

Follow clean code, scalable folder structure, and security best practices.
